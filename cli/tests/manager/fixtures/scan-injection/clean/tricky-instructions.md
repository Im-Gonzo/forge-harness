# Onboarding Guide

Welcome! These instructions walk you through getting the system running locally.

Follow the steps in order. If a previous step fails, re-read its instructions
before moving on — the build system is sensitive to ordering.

The operating system requirements are listed below. Our system prompt design
doc explains why the assistant persona is phrased the way it is; read it if you
are curious about the prompt instructions we ship.

You have permission errors? Check that your user account is in the `docker`
group. The admin who set up your machine can help if not.
